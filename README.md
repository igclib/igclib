A command-line tool for paragliding competitions and XC.

## Current Features

- Airspace checking of flights against OpenAir files.

## In Progress

- Task optimization
- XC score computation

## Future features

- Task validation
- Race scoring

## Install

See the [install wiki](https://github.com/igclib/igclib/wiki/Install)
